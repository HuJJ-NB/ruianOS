# HuJJOS-F

## 项目介绍

本项目参考rCore，为个人学习项目。

## 系统特性

批处理系统。

## 安装运行

* 工具安装请参考rCore文档，https://github.com/rcore-os/rCore-Tutorial-v3。

* 在项目根目录或者`os`目录下`make run`以运行本项目。

* 根目录下`make run-log`或者`make run FLAGS=--features\ LOG_DEBUG`开启一些调试信息的输出，后者也可以在`os`目录中使用。
